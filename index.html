Welcome

I made this site for class and went with games because it was the easiest topic for me to talk about without making it feel forced. The main point was to build a basic website, publish it online, and track how people move through the pages.

It is a pretty simple site overall, but it still has everything needed for the project like multiple pages, forms, tracked clicks, navigation, and time spent on each page. I also wanted it to feel a little more personal than just a default layout.
